A comprehensive SEO-optimized tool for US to France relocation guidance with automatic weekly updates and rich snippet support.

== Description ==

The France Relocation Assistant is a free, comprehensive resource for Americans planning to relocate to France. Built with SEO best practices for maximum visibility in search results.

**Topics Covered:**

* **Visas & Immigration** - Visitor visa, work visa, Talent Passport, spouse visa, OFII validation
* **Property Purchase** - French buying process, notaire role, mortgages, compromis de vente
* **Healthcare** - PUMA enrollment, Carte Vitale, mutuelle options, private insurance
* **Taxes** - 183-day rule, French tax residency, US-France tax treaty, FBAR, FATCA
* **Driving** - License exchange (by state), car registration, importing vehicles
* **Shipping & Pets** - Customs process, duty-free imports, pet relocation requirements
* **Banking** - Opening French accounts, FATCA compliance, required documents
* **Settling In** - Utilities, mairie registration, French bureaucracy tips

**Key Features:**

* **Side-by-Side Interface** - Navigation on left, content preview on right
* **Conversation Context** - Follow-up questions understand previous topic
* **183-Day Counter** - Track France/US presence for tax residency
* **SEO-Optimized** - Built-in schema markup for rich snippets
* **Mobile-Responsive** - Works on all devices
* **Braun-Inspired Design** - Clean, functional, timeless aesthetic

**SEO Features:**

* JSON-LD structured data (Schema.org)
* FAQ schema for rich snippet eligibility (8 high-value questions)
* HowTo schema for property purchase process
* Article schema with proper markup
* Open Graph and Twitter Card support
* Compatible with Yoast SEO and Rank Math
* Target keyword integration

== Installation ==

1. Upload the `france-relocation-assistant` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the shortcode `[france_relocation_assistant]` to any page or post
4. (Recommended) Install Yoast SEO or Rank Math for additional SEO features

== Usage ==

**Basic Shortcode:**
`[france_relocation_assistant]`

**SEO Best Practices:**
* Create the page with an SEO-friendly URL like `/france-relocation-guide/`
* Add the shortcode to a page with a descriptive title
* Use Yoast SEO or Rank Math for additional meta tag control
* Link to the page from related blog content

== SEO Schema Markup ==

The plugin automatically generates the following structured data:

**FAQ Schema (8 Questions):**
* What visa do I need to move to France from the USA?
* Can Americans buy property in France?
* How does French healthcare work for American expats?
* What is the 183-day rule for French tax residency?
* Can I exchange my US driver's license in France?
* How much money do I need to move to France?
* Can I work in France on a visitor visa?
* How do I open a French bank account as an American?

**HowTo Schema:**
* Step-by-step property purchase process
* Enables how-to rich snippets in search

Test your schema at: https://search.google.com/test/rich-results
